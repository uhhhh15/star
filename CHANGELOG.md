# Changelog

All notable changes to this project will be documented in this file.

## [2.2.0] - 2025-11-6

### 🚀 优化 (Optimization)

1.  **缓存优化**：优化了缓存机制，以解决在特殊情况下偶现 “`该收藏被删除或不存在`” 的问题。
2.  **UI 交互优化**：在暗色主题下，固定了预览模式右侧的 `👁️` 图标，使其悬停和点击更加稳定方便。

### 🎨 修改 (Changes)

1.  **视觉调整**：在暗色主题的“查看上下文”界面中，将 **user 气泡框** 的背景色调整为紫罗兰色，以增强辨识度并提升视觉美感。
2.  **默认值提升**：将每页显示的收藏数量默认值从 5 条提升至 **10** 条。
    > 温馨提示：您仍然可以在设置页面通过 `修改显示收藏数` 选项自定义此数量。
3.  **分页功能重构**：对收藏面板底部的页码导航进行了全面优化：
    *   **视觉美化**：界面会根据当前主题自动适配，更加美观。
    *   **交互优化**：改进了翻页跳转和加载逻辑，操作更流畅。
    *   **新增“跳转至指定页”**：现在，您只需 **双击** 当前页码，即可输入目标页码并快速跳转！

### 🛠️ 修复 (Fixes)

1.  修复了在 **设置页面** 中 `修改上下文数量` 选项不生效的 BUG。现在您可以自由设定“查看上下文”时展示的数量（默认为 3 条：当前对话及前后各一条）。
2.  修复了在 **设置页面** 打开时，按 `Esc` 键会错误地关闭整个收藏面板而非设置页面的 BUG。现在 `Esc` 键将正确地关闭当前打开的设置页面。

### ✨ 新增 (New Features)

1.  在 **设置页面** 新增了 `修改显示收藏数` 选项，您可以自定义每页显示的收藏条目数量（默认为 10 条）。
    > **如何进入设置页面？** 长按收藏面板左上角的头像即可。
### 更新失败

- **如果更新失败，可尝试重新卸载再次安装**：https://github.com/uhhhh15/star.git
